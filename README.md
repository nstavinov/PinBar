# PinBar Trading System – Статус проекта

Система для тестирования и торговли стратегией **PinBar** на Python.  
Ниже представлен обзор структуры проекта и текущий статус каждого компонента.

---

## 🔹 Основные файлы

| Файл | Описание | Статус |
|------|----------|--------|
| `config.py` | Конфигурационный файл проекта, общие настройки | 🟢 Готово |
| `config_local.py` | Локальные ключи API (не пушится на GitHub) | 🟢 Готово |
| `trading_symbols.txt` | Список торговых инструментов | 🟢 Готово |

---

## 🔹 Модуль Backtest

| Файл | Описание | Статус |
|------|----------|--------|
| `backtest/run_backtest.py` | Скрипт запуска бэктеста на нескольких инструментах | 🟡 Частично (нужны исторические данные) |

---

## 🔹 Модуль Strategy

| Файл | Описание | Статус |
|------|----------|--------|
| `strategies/pinbar_strategy.py` | Реализация стратегии PinBar | 🟡 Базовый каркас |

---

## 🔹 Модуль Live Trading

| Файл | Описание | Статус |
|------|----------|--------|
| `live/trade_binance.py` | Работа с API, WebSocket, управление позициями | 🔴 В разработке |

---

## 🔹 Утилиты (utils)

| Файл | Описание | Статус |
|------|----------|--------|
| `downloader.py` | Загрузка исторических данных | 🟢 Готово |
| `order_manager.py` | Управление ордерами | 🔴 В разработке |
| `position_manager.py` | Управление позициями | 🔴 В разработке |
| `risk_manager.py` | Расчёт риска, SL, TP | 🟡 Частично |
| `indicators.py` | SMA и функции распознавания PinBar | 🟡 Частично |
| `metrics.py` | Сбор статистики и метрик бэктеста | 🟡 Частично |
| `logger.py` | Логирование действий системы | 🟢 Готово |

---
